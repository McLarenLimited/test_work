# Последовательность 1223334444…

Программа на Python выводит первые **n элементов** последовательности,  
в которой каждое число повторяется столько раз, чему оно равно:

1 → 1 раз  
2 → 2 раза  
3 → 3 раза  
4 → 4 раза  
и т.д.

## Ввод
Одно целое положительное число `n`.

## Вывод
Первые `n` элементов последовательности, выведенные в одну строку.

## Обработка ошибок
- Если введено не число — выводится сообщение об ошибке
- Если число меньше либо равно нулю — выводится сообщение об ошибке
